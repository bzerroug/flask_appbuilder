Meteo for dataware
==================

- Installation:

    # Create and enter in a virtualenv
    virtualenv /tmp/venv
    source /tmp/venv/bin/activate

    git clone https://github.com/bzerroug/flask_appbuilder
    cd flask_appbuilder
    pip install -e .

- Run:

    # Assuming you're in the venv
    dataware-meteo
